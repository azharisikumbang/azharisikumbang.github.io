<!--kg-card-begin: html--><p>Bagaimana sih install ubuntu ? Mungkin pertanyaan itu akan anda ucapkan ketika ingin mulai mengenal ubuntu dan ingin menginstallnya. Sebelumnya ubuntu adalah<br />
<a name="more"></a>salah satu sistem operasi milik linux yang dengan basis debian. Sebenarnya ubuntu dirancang untuk penggunaaa pribadi, namun sekarang ini ubuntu telah tersebar dan dipakai secara luas. Aduh, jadi basa basi jadinya, heheheh. tak perlu berlama lama langsunag saja kita melanjutkan ke <b>Instalasi Ubuntu Server Itu Mudah. </b></p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/ubuntuhero.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/ubuntuhero.jpg" width="320" height="250" border="0" /></a></div>
<h3><b>Persiapan Instalasi</b></h3>
<p>Yah, sebagai langkah awal dalam instalasi ubuntu, tentu kita perlu file instalasi nya heheh agan bisa donlot di internet. Dan pada kesempatan ini saya menggunakan Virtual Machine VBox untuk instalasi ubuntu saya.</p>
<p><b>Langkah #1 : </b>Pemilihan Bahasa, Pilih saja<b> English</b><br />
<b>Langkah #2 : </b>Keyboard Detected, pilih saja <b>No</b> untuk mempercepat instalasi.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-43-49-631.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-43-49-631-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #3 : </b>Pemilihan Asal Produksi Keyboad, pilih saja <b>English/UK</b></p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-43-57-556.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-43-57-556-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #4 : </b>Pemilihan pencocokan keyboard dengan OS, masih pilih English.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-44-07-969.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-44-07-969-300x200.jpg" width="320" height="213" border="0" /></a><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-44-01-013.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-44-01-013-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #5 : </b>Konfigurasi Jaringan, anda bisa saja tidak melakukan langkah ini dan jika ingin melekati langkah konfigurasi network, pilih <b>Do not Configure The Network At This Time. </b>Tapi pada sesi instalasi ubuntu ini saya meilih konfigurasi langsung secara manual maka pili<b> Configure The Network Manually</b></p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-46-25-170.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-46-25-170-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #6 : </b>Masukkan IP Address, Netmask, Gateway Dan DNS. Dalam kongirusari ini saya memakai  ip : 192.168.11.92 netmask : 255.255.255.0 gateway : 192.168.11.50 dan dns 8.8.8.8 . dan anda bisa mengisi bagian ini sesuai koneksi anda.</p>

<p><b>Langkah #7 : </b>Pembuatan Full Name User Ubuntu , anda bisa buat sesuai kehendak anda dan saya memilih <b>&#8220;kumbang&#8221; </b>sebagai full usernya.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-23-778.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-23-778-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #8 : </b>Pembuatan user login, isi sesuai kehendak anda dan selalu ingat karena akun ini aka ndigunakan sebagai user login. dan saya jgua memakai <b>&#8220;kumbang&#8221;</b></p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-28-757.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-28-757-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #9 : </b>Pemasukan Password</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-39-425.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-39-425-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #10 : </b>Re-type Password. Ketik ulang password anda.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-46-482.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-46-482-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #11 : </b>Jika anda menggunakan password yang pendek maka akan mucul dialog beriku, pilih saja yes untuk menyetujui penggunaaan password yang singkat.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-53-058.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-53-058-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #12 : </b>Enkripsi Home Direktori. Pilih Yes untuk mengenkripsi direktori user anda.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-58-316.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-48-58-316-300x200.jpg" width="320" height="213" border="0" /></a></div>
<p><b>Langkah #13 : </b>Partisi Hardisk. Anda bisa memilih mana saja, namun saya lebih suka mempartisi manual. Pilih <b>Manual.</b></p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-09-909.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-09-909-300x223.jpg" width="320" height="238" border="0" /></a></div>
<p><b>Langkah #14 : </b>Pilih Hardisk yang akan dipartisi. Karena saya memakai harddisk bawaan VBOX maka saya meilih VBOX punya untuk di partisi.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-17-911.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-17-911-300x223.jpg" width="320" height="238" border="0" /></a></div>
<p><b>Langkah #15 : </b>Pembuatan partisi baru. Pilih <b>Yes. </b>Tapi dengan catatan jika ingin mempartisi pastikan data data anda dikomputer telah terbackup karena seluruh data akan dihapus.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-33-261.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-33-261-300x223.jpg" width="320" height="238" border="0" /></a></div>
<p><b>Langkah #16 : </b>Pilih pilihan tepat dibawah partisi hardisk anda tadi.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-43-569.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-43-569-300x223.jpg" width="320" height="238" border="0" /></a></div>
<p><b>Langkah #17 : </b>Pilih <b>Automatically Partition The Free Space. </b>Untuk membuat partisi direktori otomatis pada partisi kita sebelumnya.</p>
<div style="clear: both; text-align: center;"><a style="margin-left: 1em; margin-right: 1em;" href="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-50-698.jpg"><img loading="lazy" src="/content/images/wordpress/2017/04/bandicam-2B2017-04-27-2B03-51-50-698-300x223.jpg" width="320" height="238" border="0" /></a></div>
<p><b>Langkah #18 : </b>Langkah Partisi selesai. Pilih <b>Finish The Partition And Write The Changes to Disk</b></p>

<p><b>Langkah #19 : </b>Pilih <b>Yes. </b>dan alnjutkan instalasi ubuntu,</p>

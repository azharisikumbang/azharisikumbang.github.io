Sharing File Windows

posted at 2016-09-09

Sharing atau berbagi source/sumberdaya adalah suatu fasilitas yang kita bisa gunakan dalam suatu jaringan. Contoh sumber daya seperti file, document, video, foto dan sebagainya dapat kita bagikan dengan komputer lainnya dalam suatu koneksi jaringan, seperti

jaringan LAN. Suatu sumber daya dapat dibagikan dari komputer kita ke suatu jaringan dapat kita lakukan dengan singkat dan tidak memakan waktu lama. Lain halnya apabila kita menggunakan flashdisk atau media penyimpan lainnya.

 
Berikut adalah Tuorial Sharing File di Windows :
Langkah #1 :  Buka Windows Explorer, Cari dan pilig file/folder yang akan di share, disini saya akan share folder “Ilham” dengan Klik kanan ->> pilih share with ->> Pilih Nobody,
 

Langkah #2 :  Setelah itu pilih Change Sharing Permissions,

 
Langkah #3 :  Lalu, Klik panah dan pilih “Everyone” ->> lalu ADD,

Langkah #4 :  Setelah di ADD, Klik list Everyone dan pilih share,

Langkah #5 :  Kemudian, Pilih yes atau no, terserah anda
Langkah #6 :  Terakhir, Klik file yang akan di share dan Klik Done.

Langkah #7 :  Untuk Melihat apakah file/folder telah di share anda bisa mengetik ip dari komputer yang melakukan share pada browser Windows Explorer. Disini saya menggunakan ip 192.168.181.44 , maka kita mengetik 192.168.181.44 di browser Windows Explorer.

 
Atau, dapat dilihat dengan cara lain, yakni dengan menklik network pada windows explorer, Pilih dan Klik Nama PC kita, Maka File/Folder yang kita share akan muncul

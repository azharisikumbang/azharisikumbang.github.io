Instalasi Ubuntu Server Itu Mudah

posted at 2017-04-28

Bagaimana sih install ubuntu ? Mungkin pertanyaan itu akan anda ucapkan ketika ingin mulai mengenal ubuntu dan ingin menginstallnya. Sebelumnya ubuntu adalah

salah satu sistem operasi milik linux yang dengan basis debian. Sebenarnya ubuntu dirancang untuk penggunaaa pribadi, namun sekarang ini ubuntu telah tersebar dan dipakai secara luas. Aduh, jadi basa basi jadinya, heheheh. tak perlu berlama lama langsunag saja kita melanjutkan ke Instalasi Ubuntu Server Itu Mudah. 

Persiapan Instalasi

Yah, sebagai langkah awal dalam instalasi ubuntu, tentu kita perlu file instalasi nya heheh agan bisa donlot di internet. Dan pada kesempatan ini saya menggunakan Virtual Machine VBox untuk instalasi ubuntu saya.

Langkah #1 : Pemilihan Bahasa, Pilih saja English
Langkah #2 : Keyboard Detected, pilih saja No untuk mempercepat instalasi.

Langkah #3 : Pemilihan Asal Produksi Keyboad, pilih saja English/UK

Langkah #4 : Pemilihan pencocokan keyboard dengan OS, masih pilih English.

Langkah #5 : Konfigurasi Jaringan, anda bisa saja tidak melakukan langkah ini dan jika ingin melekati langkah konfigurasi network, pilih Do not Configure The Network At This Time. Tapi pada sesi instalasi ubuntu ini saya meilih konfigurasi langsung secara manual maka pili Configure The Network Manually

Langkah #6 : Masukkan IP Address, Netmask, Gateway Dan DNS. Dalam kongirusari ini saya memakai  ip : 192.168.11.92 netmask : 255.255.255.0 gateway : 192.168.11.50 dan dns 8.8.8.8 . dan anda bisa mengisi bagian ini sesuai koneksi anda.

Langkah #7 : Pembuatan Full Name User Ubuntu , anda bisa buat sesuai kehendak anda dan saya memilih “kumbang” sebagai full usernya.

Langkah #8 : Pembuatan user login, isi sesuai kehendak anda dan selalu ingat karena akun ini aka ndigunakan sebagai user login. dan saya jgua memakai “kumbang”

Langkah #9 : Pemasukan Password

Langkah #10 : Re-type Password. Ketik ulang password anda.

Langkah #11 : Jika anda menggunakan password yang pendek maka akan mucul dialog beriku, pilih saja yes untuk menyetujui penggunaaan password yang singkat.

Langkah #12 : Enkripsi Home Direktori. Pilih Yes untuk mengenkripsi direktori user anda.

Langkah #13 : Partisi Hardisk. Anda bisa memilih mana saja, namun saya lebih suka mempartisi manual. Pilih Manual.

Langkah #14 : Pilih Hardisk yang akan dipartisi. Karena saya memakai harddisk bawaan VBOX maka saya meilih VBOX punya untuk di partisi.

Langkah #15 : Pembuatan partisi baru. Pilih Yes. Tapi dengan catatan jika ingin mempartisi pastikan data data anda dikomputer telah terbackup karena seluruh data akan dihapus.

Langkah #16 : Pilih pilihan tepat dibawah partisi hardisk anda tadi.

Langkah #17 : Pilih Automatically Partition The Free Space. Untuk membuat partisi direktori otomatis pada partisi kita sebelumnya.

Langkah #18 : Langkah Partisi selesai. Pilih Finish The Partition And Write The Changes to Disk

Langkah #19 : Pilih Yes. dan alnjutkan instalasi ubuntu,

Langkah #20 : HTTP Prosy kosongkan saja dan pilih Continoue.

Langkah #21 : Pemilihan Upgrade system data. Pilih saja No Automaticly Update.

Langkah #21 : Centang OpenSSH Server. Tapi anda bisa pilih sesuai keperluan. Dan pilih Continue.

Langkah #22 : Install Grub Loader. Pilih Yes.

Langkah #23 : Pilih Contonie pada PAM Configuration untuk menyelesaikan instalalsi

Langkah #24 : Komputer akan Reboot dan masuk kehalaman login.

Langkah #25 : Instalasi selesai dan selamat pusing dengan command line hehehehe

Benarkan Instalasi Ubuntu Server Itu Mudah ? Ingat ! Instalasi bisa anda customisasi sesuai kebutuhan anda tergantu selera anda. Langkah diatas hanya langkah yang saya pelajari dan tentu sesuai selara saya. Hehehe . Dan jika ada kebingungan silahkan agan komen dibawah !!
Sekian tentang Instalasi Ubuntu Server Itu Mudah semoga bermanfaat !!

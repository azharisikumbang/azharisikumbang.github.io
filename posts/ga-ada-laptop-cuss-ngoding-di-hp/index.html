Ga ada laptop? Cuss ngoding di Hp

Assalamualaikum

Kali ini kita ga bakal komding di laptop, bukan buat gaya gaya an, tapi emang laptop rusak, dan kebetulan dapat tugas dari kampus untuk yaitu pemrograman dasar, serta dapat request juga.

Sekedar info, tugas yang saya dapat cukup mudah, ada beberapa task yang continue gitu, dimulai dari nge print data, pengenalan variabel dan hingga operasi matematika, tapi yang bakal dibahas beberapa saja, yaitu ngeprint data dan task agak sulit, yaitu operasi matematika untuk rumus segitiga.

Okee, ga usah basa basi, sebelum mulai ada beberapa hal yang dibutuhkan, pertama handphone dan koneksi internet, usahakan pake paket hadiah kemdikbud biar lebih berguna hehe..

Kenalan dengan repl.it

Ini bukan endorse.. cuman untuk saat ini repl.it memang memenuhi kebutuhan, selain kelengkapannya yajg mendukung beberapa bahasa program, disediakan juga terminal untuk eksekusi program kita, jadi lumayan joss dari pada sekedar tekan run truss compile dibeberapa IDE online.

Buat akun

Pertama, kunjungi https://repl.it/ tidka usah terintimidasi dengan tampilan dari repl.it . Untuk mendaftarkan akun silahkan cari Sign Up, kemudia kita akan dialihkan ke halaman registrasi, silahkan daftarkan akun seperti biasa, tapi saran biar cepat daftar pake akun google aja karena lumrahnya setiap hp sudah dilengkapi akun google.

Membuat Project

Setelah mendaftarkan akun kita akan dialihkan ke dashboard milik kita, jangan terintimidasi, cukup cari tanda + dipojok kanan atas atau “create new repl”,

Kemudian akan muncul dialog berikut

Untuk bahasa silahkan disesuaikan, piliha sesuai keinginan, tapi karena task yang diberikan mengharuskan memakai Java, jadi saya pilih java.

Sedangkan untuk nama project atau repl name, silahkan disesuaikan juga, dan karena task yang saya terima mengharuskan namanya Lesson01

Dan sekarang pilih create repl

Waktunya Koding

Eh sabar dulu, sebenarnya mulai dari sini silahkan disesuaikan dengan apa yang akan anda buat, contoh berikut hanya berdasarkan tugas saya saja

Pertama kita akan dihadapkan ke interface seperti berikut.

Seperti biasa kita akan diberikan class Main, tidak usah diganti gugat, biarkan saja, karena task tidak mewajibkan kita berinteraksi dengan itu.

Pada task ini, kita diharuskan membuat package bernama com.program, jadi sesuatu behavior dari java hal ini mewakili folder.

Untuk itu, kita harus membuat folder com/tutorial, caranya cukup mudah, silahkan ketuk tulisan Files warna biru, dan akan memunculkan dialog modal seperti berikut

Perlu diketahui untuk membuat file pilih logo yang dilingkari merah, sedangkan folder logo yang dilingkari hijau

Hebat.java

Untuk file pertama kita beri nama Hebat.java, ingat sensitifitas huruf, pastikan jadi huruf kecil dan besar itu berbeda.

Untuk kodingnya sederhana, seperti ritual programmer pda umumnya, yaitu Hello, world

Sehingga kodenya jadi seperti berikut

catatan : jika normalnya kita melakukan compiling adalah menggunakan javac, di tutorial ini tidak akan kita lakukan compiling, melainkan sekedar run dengan java, dikarenakan target pembaca adalah orang yang baru pertama kali kenak java dan akan sedikit memusingkan dengan begitu banyak istilah baru bagi pembaca. Tapi jika ingin compiling, silahkan sesuaikan codenya, tekan tombol run, dan jalankan java classnya

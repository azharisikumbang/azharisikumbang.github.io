Cara Membuat Loopback Interface Adapter Windows 7

posted at 2016-09-09

Loopback adapater adalah sebuah network adapter yang divirtualkan. Jadi bukan hanya sistem operasi yang divirtualkan, kali ini kira juga akan memvirtual kan network adapter yang dinamakan Loopback Adapter. Loopback Adapter biasanya digunakan untuk melakukan
testing networking lewat software software vitrual machine.

Baiklah berikut langkah langkah nya :
Langkah #1 : Masuk ke menu Run, dengan cara Tekan Logo Windows + R dan ketikkan hdwwiz.exe dan tekan Enter.

Langkah #2 : Setelah itu akan muncul kotak dialog Add Hardware, Klik Next.
Langkah #3 : Pilih “Install the hardware that I manually select from list (Advanced)” dan Next.

Langkah #4 : Akan muncul kotak dialog berikutnya, Pilih Network Adapter, dan Next.
Langkah #5 : Pada kotak pilihan kiri, pilih  Microsoft. dan pada kotak pilihan kanan pilih Microsoft Loopback Adapter. dan Next.

Langkah #6 : Pilih dan Kilik Next.
Langkah #6 : Selesai, pilih dan klik Finish.

Demikianlah tentang Cara membaut Loopback Adapter di Windows 7..
